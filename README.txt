***** Instructions to setup the virtual environment (venv) and run the program *****


1. FAST & EASY: 	Go to the Asuntosijoituslaskuri folder and type "cmd" in the bar box above the window -> CMD opens with right PATH
   SLOW:		Go to Command Prompt on Windows and set path to the Asuntosijoituslaskuri folder (cd PATH_name) OR
   
2. Type "asunto_env\Scripts\activate" to command prompt to ACTIVATE the virtual environment

3. Type "laskuri.py" to RUN the program.

4. Type "deactivate" to exit the virtual environment (venv)
